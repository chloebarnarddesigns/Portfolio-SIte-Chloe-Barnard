Project Retrospective

What Did You Learn?

This was my first project designing my own end-to-end case study. I learned about the design process particularly the user interface design process. I learned about UX and UI methodologies including contextual research, user flows, information architecture, wireframes, icons, style guides, and prototypes.

What Challenges Did You Encounter?

Although, I had practiced using Adobe XD to create screens this was my first time desiging my own. I realized how important it is to develop a organization system for files expecially given the high number of iterations. I found designing icons to be challenging with their small size.

What Are the Next Steps?

The next steps are to prepare and to hand over the design assets to engineers. I would conduct more usability testing after that point.
